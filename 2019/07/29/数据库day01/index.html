    数据库概念

    1. 传统存储数据的方式
    1. 用大脑记，容易忘
    2. 用笔和纸记，麻烦，查起来困难
2. 如何解决
使用数据库记录数据，存储方便，查询方便
3. 概念
        1. 数据
        2. 现实世界中一切可以被字符和数字描述的事物都可以称为数据
        3. 关系型数据
               数据有描述，数据与数据之间有关系
        4. 二维数据表
               由数据行和数据列组成的表格称为二维数据表
        5. 关系型数据库
               存储二维数据表的仓库，Oracle安装后自带一个名字叫orcl的数据库
        6. 数据库服务
               我们执行的setup.exe程序就是安装数据库服务的,可以在我的电脑-->管理-->服务里面查询相关的服务信息
               其中最重要的两个服务：
                    1. OracleServiceORCL  这个是orcl数据库的服务
                2. OracleOraDb11g_home_1TNSListener   这个是数据库监听服务,它负责设置数据库监听的IP和端口,它不影响本地数据库连接
        7. 数据库客户端
            使用win+r键输入cmd打开一个黑窗口,这里可以使用命令连接和操作数据库
        8. 数据库图形化管理工具
            就是Sql Developer这个工具,这是Oracle公司自己开发的免费工具,还有其他的第三方工具,如PL/SQL Developer
                解压SqlDeveloper之后,打开,将里面的sqldeveloper.exe程序发送到桌面快捷方式,由于这个工具是用Java开发的,所有需要配置Java环境
        9. 知名的数据库厂商
                1. Oracle        oracle公司
                2. Sql Server    微软公司
                3. MySql         oracle公司
                4. DB2           IBM公司

        10. 软件架构体系
                B/S   Browser/Server 浏览器服务器架构
                C/S   Client/Server  客户端服务器架构
                Oracle采用的是C/S模式
        11. 用户账户介绍
                    1. Sys      超级管理员  密码123456
                 2. System   管理员      密码123456
                 3. 普通用户(scott是普通用户中的一个)
            scott用户默认是被锁定的,且密码以及过期了
           在安装的时候可以解锁和修改密码
            在dos里面使用 sqlplus scott/tiger 回车
            出现SQL>_   说明scott用户可以使用
            如果在安装的时候未解锁:
            1. 使用 sqlplus / as sysdba 登录
            2. alter user scott account unlock; 解锁
            3. alter user scott identified by tiger; 重置密码为tiger
     12. SqlPlus客户端本地连接(客户端和服务在同一个操作系统中)
    sqlplus 用户名/密码
     13. SqlPlus客户端远程连接(客户端和服务不在同一个操作系统中)
    在虚拟机中的dos里面输入ipconfig 查看虚拟机的ip地址
    IP Address 后面的 192.168.58.128就是虚拟机ip地址
    在本机的dos里面输入
    sqlplus scott/tiger@//192.168.58.128:1521/orcl
